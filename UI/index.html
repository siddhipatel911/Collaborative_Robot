About this project

This demo shows a simple proof-of-concept for a collaborative robot that watches a placement zone using an overhead camera, detects coloured blocks, and performs pick-and-place operations. The system adapts pace and behaviour according to user presence and a simple mood/personality model. Use this page to preview the camera feed while running the robot software.

The stream is provided by collaborative_demo.py. Make sure it is running to see live camera feed with detection overlays.
